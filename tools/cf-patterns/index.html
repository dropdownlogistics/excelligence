Cell / Column VALUE
ГўвҖ“Вҫ

Duplicate Detection
Flag all instances of a value that appears more than once
Column VALUE
ГўвҖ“Вҫ
When to Use
Invoice auditing ГўвӮ¬вҖқ catch duplicate payment submissions

Commission reconciliation ГўвӮ¬вҖқ flag duplicate transaction entries
Multi-column duplicate check ГўвӮ¬вҖқ COUNTIFS handles this better
Scope Guidance
Lock the column reference ($A$2:$A$1000) but leave the row in the formula relative (A2). This lets the rule walk down the column while always counting against the full range.
=COUNTIF($A$2:$A$1000,A2)>1 =COUNTIF($A$2:A2,A2)>1
ГўЕЎВ  The built-in Duplicate Values rule highlights all occurrences including the first. If you need to keep the first instance clean and only flag the repeat, use the expanding range variant above.

Overdue / Date Past
Highlight dates that have passed today or are approaching
Column DATE
ГўвҖ“Вҫ
When to Use
Action item tracking ГўвӮ¬вҖқ surface past-due items automatically
Contract renewals ГўвӮ¬вҖқ flag expirations within 30 days
Audit findings ГўвӮ¬вҖқ highlight items past remediation deadline
Static date comparisons ГўвӮ¬вҖқ just use a helper column instead
Scope Guidance
TODAY() recalculates on every open. Apply to the full date column. Chain two rules ГўвӮ¬вҖқ overdue (red) and approaching (amber) ГўвӮ¬вҖқ in that priority order so overdue always wins.
=AND(C2<TODAY(),C2<>"") =AND(C2>=TODAY(),C2<=TODAY()+7)
ГўЕЎВ  Always include the blank check (C2<>"") on the overdue rule. Without it, empty date cells evaluate as 0, which is before TODAY(), and your whole empty column turns red.

Full Row Highlight
Shade an entire row when a condition in one column is true
Row VALUE
ГўвҖ“Вҫ
When to Use
Exception reporting ГўвӮ¬вҖқ shade entire row when flagged
Priority tiers ГўвӮ¬вҖқ visually separate high-priority records
Status filtering ГўвӮ¬вҖқ make closed/complete rows recede visually
Tables with many columns ГўвӮ¬вҖқ test performance before applying to 50+ columns
Scope Guidance
Apply to the full row range (e.g., $A2:$Z2). The trigger column must be column-locked ($E2) ГўвӮ¬вҖқ lock the column letter, leave the row number relative so it walks down each row.
=$E2="HIGH" =$F2>1000 =AND($E2="OPEN",$F2>TODAY())
ГўЕЎВ  The most common mistake: forgetting to lock the column ($E2 not E2). Without the $, the rule shifts horizontally across columns and evaluates against the wrong column for every cell.

3-Color Scale (Heat Map)

ГўвҖ“Вҫ
When to Use
